# 经济学人文章字体

内容的主要字体为 MiloTE.

## 安装字体到多看阅读app

1. 下载字体到电脑上
2. 用多看阅读打开epub格式的杂志
3. 调出书籍阅读页面菜单, 选择"Aa" 菜单 -> "字体" -> "更多字体", 选择"英文字体", 然后点击右上角的 "+" 按钮
4. 此时会得到一个地址, 在浏览器中输入该地址
5. 然后将下载到的 ttf 字体拖到浏览器里面, 上传成功之后字体就安装到多看阅读中了
6. 在多看阅读中选择 MiloTE 字体即可.
